\doxysubsection{Detailed Description}
A class that manages blocking of IP addresses by prefix. 

The \doxylink{class_i_p_range_blocker}{IPRange\+Blocker} class allows for blocking specific IP address prefixes, checking if incoming requests are from blocked addresses, and rejecting them if necessary.
