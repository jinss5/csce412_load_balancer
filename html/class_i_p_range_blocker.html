Detailed Description

A class that manages blocking of IP addresses by prefix.

The IPRangeBlocker class allows for blocking specific IP address prefixes, checking if incoming requests are from blocked addresses, and rejecting them if necessary.

void IPRangeBlocker::blockIPPrefix ( const std::string & prefix)

Adds an IP prefix to the list of blocked addresses.

Parameters
prefixThe IP prefix to block (e.g., "192.168.0.").

This function allows you to block an entire range of IP addresses by specifying a prefix. For example, blocking "192.168.0." will block all addresses that start with "192.168.0.".
